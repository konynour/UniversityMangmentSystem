University Management System (UMS)

A modern ASP.NET MVC-based University Management System designed with a bold **Neo-Brutalism UI** style.
The system provides centralized management for departments, instructors, courses, and trainees through a fast and visually distinctive dashboard interface.

---

# 🚀 Features

## 📚 Department Management

* Create departments
* Edit department data
* Delete departments
* View all university sectors

## 👨‍🏫 Instructor Management

* Manage academic staff
* View instructor information
* Assign instructors to departments

## 📖 Course Management

* Organize courses
* Manage curriculum data
* Link courses with departments

## 🎓 Trainee / Student Management

* Track student records
* Manage grades
* View trainee information

---

# 🎨 UI Design

This project uses a **Neo-Brutalism Design System** featuring:

* Bold typography
* Thick borders
* Hard shadows
* High-contrast colors
* Block-based layout
* Responsive dashboard cards

Example UI philosophy:

> "LOUD. FAST. BRUTAL."

The interface intentionally avoids soft modern minimalism and focuses on strong visual identity and usability.

# 📦 Core Modules

The system includes four main management modules:

| Module      | Description                        |
| ----------- | ---------------------------------- |
| Departments | Manage university sectors          |
| Instructors | Handle academic staff data         |
| Courses     | Organize curriculum                |
| Trainees    | Track student information & grades |

These modules are accessible directly from the dashboard interface.

# 💻 Getting Started

## 1) Clone the Repository

```bash
git clone <repository-url>
```

---

## 2) Open the Project

Open the solution using:

* Visual Studio
* Rider
* VS Code (with C# extensions)

---

## 3) Restore Packages

```bash
dotnet restore
```

---

## 4) Run the Application

```bash
dotnet run
```

Or run directly from Visual Studio.
